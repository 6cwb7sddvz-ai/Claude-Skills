# Web Research

Use this skill when researching a topic across multiple sources.

## Principles
- Prefer primary and authoritative sources over aggregators.
- Cross-check claims across at least two independent sources.
- Note publication dates; prefer current information.
- Separate facts from opinion and flag uncertainty.
- Track and cite every source used.

## Workflow
1. Clarify the question and what a good answer looks like.
2. Search broadly, then narrow to the best sources.
3. Extract key facts with citations.
4. Synthesize into a concise, structured summary.
5. List sources and note any gaps or conflicts.
