      # Data Analysis

      Use this skill for exploring datasets and producing insights.

      ## Capabilities
      - Load data from CSV, Excel, JSON, and SQL
        - Clean: handle missing values, types, duplicates, outliers
          - Aggregate, group, join, and pivot
            - Visualize distributions, trends, and correlations

              ## Recommended tools
              - pandas and numpy for manipulation
              - matplotlib or seaborn for plots
              - jupyter for exploratory work

              ## Workflow
              1. Understand the question before touching the data.
              2. Profile the data: shape, types, missingness, ranges.
                3. Clean and transform, documenting each assumption.
                  4. Analyze, visualize, and state findings with caveats.
