# Excel (xlsx)

Use this skill when working with spreadsheets.

## Capabilities
- Read and write cell values across multiple sheets
- Apply formulas, number formats, and conditional formatting
- Build charts and pivot tables
- Clean and analyze tabular data

## Recommended tools
- openpyxl for reading/writing and formatting
- pandas for data analysis and bulk operations

## Workflow
1. Load the workbook and inspect sheet names and headers.
2. Use pandas for heavy data work, openpyxl for formatting.
3. Save and verify formulas evaluate correctly on open.
