# PDF Toolkit

Use this skill when working with PDF files.

## Capabilities
- Extract text and tables from PDFs
- Fill and flatten PDF form fields
- Merge multiple PDFs or split a PDF into pages
- Generate new PDFs from HTML or Markdown

## Recommended tools
- pypdf for reading, merging, splitting
- pdfplumber for text and table extraction
- reportlab or weasyprint for generation

## Workflow
1. Identify the task (extract, fill, merge, split, generate).
2. Choose the appropriate library above.
3. Validate output by re-opening the PDF and checking page count and content.
