# PowerPoint (pptx)

Use this skill when creating or editing slide decks.

## Capabilities
- Create slides from layouts and templates
- Add titles, bullet content, images, and shapes
- Insert charts and tables
- Edit existing presentations

## Recommended tools
- python-pptx for programmatic slide creation and editing

## Workflow
1. Choose a template or blank presentation.
2. Add slides using named layouts for consistency.
3. Keep text concise; verify slides render without overflow.
