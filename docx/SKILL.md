# Word (docx)

Use this skill when creating or editing Word documents.

## Capabilities
- Create documents with headings, paragraphs, and styles
- Insert tables, images, headers/footers, and page breaks
- Read and modify existing .docx content
- Apply and track changes/comments

## Recommended tools
- python-docx for creating and editing documents
- docx2txt for quick text extraction

## Workflow
1. Load or create the Document object.
2. Apply consistent styles rather than manual formatting.
3. Save and re-open to verify layout renders correctly.
